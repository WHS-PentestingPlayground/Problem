# Shall We Melt the Spring?

### ⚠️ 취약점 (Vulnerabilities)

 **RCE(Remote Control Execution)**

### 📌 기술 스택 (Tech Stack)

| **분야** | **사용 기술** |
| --- | --- |
| **프레임워크** | Spring Boot 2.6.5 |
| **언어** | Java 17 |
| **웹 서버** | Tomcat 9.0.53 |
| **빌드 도구** | Gradle |

### 📝 문제 설명

| 목표 : RCE로 이어지는 컨트롤러 침투 경로를 분석하고, 사용자 입력을 통해 시스템 레벨에 도달하라.
